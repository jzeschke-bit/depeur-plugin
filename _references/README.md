# References

Dieser Ordner enthält Code, der **nicht** Teil des laufenden WordPress ist.
Er dient ausschließlich als Lesematerial für Claude Code.

## legacy-plugins/
Eigene Mini-Plugins, die in `plugins/depeur-food` konsolidiert werden sollen.
Quelle für Funktionalität, die in das neue Plugin übernommen wird.

## legacy-themes/
Bestehende Child-Theme-Stände. Quelle für Customizations, die in das neue
saubere `themes/kadence-child` übernommen werden sollen.

## plugin-references/
Drittanbieter-Code (BunnyCDN, RunCloud Hub, WP Rocket). Architektur- und
Pattern-Inspiration. **Nicht in git getrackt** (siehe .gitignore).
